# 脚本

`scripts/` 目录包含用于本地工作流和运维任务的辅助脚本。
当任务明确依赖于脚本时使用它们；否则优先使用命令行工具（CLI）。

## 规范

- 脚本是**可选的**，除非文档或发布检查清单中有提及。
- 优先使用已有的 CLI 界面（例如：认证监控使用 `openclaw models status --check`）。
- 脚本通常是主机特定的；在新机器上运行前请先阅读脚本内容。
