# TOOLS.md - 本地笔记

技能定义了工具的_工作方式_。此文件记录的是_你的_具体内容——与你的设置独一无二的东西。

## 这里写什么

例如：

- 摄像头名称和位置
- SSH 主机和别名
- TTS 首选语音
- 扬声器/房间名称
- 设备昵称
- 任何与环境相关的内容

## 示例

```markdown
### 摄像头

- living-room → 主区域，180°广角
- front-door → 入口，动作触发

### SSH

- home-server → 192.168.1.100，用户：admin

### TTS

- 首选语音："Nova"（温暖，略带英式口音）
- 默认扬声器：厨房 HomePod
```

## 为什么分开？

技能是共享的。你的设置是你的。将它们分开意味着你可以更新技能而不丢失笔记，也能共享技能而不暴露你的基础设施。
