# TOOLS.md - 用户工具笔记（可编辑）

此文件用于记录你关于外部工具和约定的笔记。
它并不定义存在哪些工具；OpenClaw 在内部提供内置工具。

## 示例

### imsg

- 发送 iMessage/SMS：描述收件人/内容，发送前确认。
- 建议发送简短消息；避免发送机密信息。

### sag

- 文字转语音：指定语音、目标扬声器/房间，及是否流式传输。

你可以添加任何你希望助手了解的本地工具链信息。
