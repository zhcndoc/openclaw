# `openclaw setup`

初始化 `~/.openclaw/openclaw.json` 及代理工作区。

相关内容：

- 入门指南：[入门指南](/start/getting-started)
- 向导：[引导](/start/onboarding)

## 示例

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

通过 setup 运行向导：

```bash
openclaw setup --wizard
```
