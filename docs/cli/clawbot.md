# `openclaw clawbot`

为向后兼容保留的旧别名命名空间。

当前支持的别名：

- `openclaw clawbot qr`（与 [`openclaw qr`](/cli/qr) 行为相同）

## 迁移

推荐直接使用现代顶层命令：

- `openclaw clawbot qr` -> `openclaw qr`
